Hermes SEO — Pipeline Audit de Contenu

Architecture v0.1 · 21 juin 2026 · 10 agents · ~70% de code réutilisé · ~12h estimées

Rappel important : saas-seo contient du code testé superficiellement. On ne copie pas aveuglément — on s'inspire de la logique et on réimplémente proprement dans Hermes avec nos standards (Pydantic, tests, fallbacks open-source).

1. Rôle du pipeline

Évaluer les pages existantes d'un site sur tous les aspects SEO / AEO / GEO / EEAT / UX et produire un rapport d'audit complet avec :

  • Un score par page et par dimension
  • Des forces et faiblesses identifiées
  • Des recommandations actionnables priorisées
  • Une roadmap de réécriture

Le pipeline ne publie rien — il diagnostique. La réécriture est gérée par le pipeline Éditorial (existant), qui reçoit l'audit_brief en entrée.

5. Connexion avec le pipeline Éditorial

Le pipeline Audit NE PUBLIE PAS. Il produit un audit_brief structuré que le pipeline Éditorial consomme pour réécrire les pages.

Pipeline Audit de Contenu

        │  audit_brief = {
        │    "page_url": "https://client.fr/faible",
        │    "scores": { seo: 45, aeo: 30, geo: 25, eeaT: 8, ux: 60 },
        │    "forces": ["H1 présent et pertinent", "Images avec alt"],
        │    "faiblesses": ["Pas de FAQ", "0 source citée", "Auteur absent"],

6. Interface utilisateur

Nouvel onglet dans la sidebar Streamlit : "Audit de Contenu"

ÉtapeCe que l'utilisateur voit
1Champ : URL du site à auditer (ou liste d'URLs)
2Bouton : "Lancer l'audit"
3Progression : barre avec les pages traitées (3/15)
4Résultat : dashboard d'audit avec scores par page, distribution, alertes
5Par page : fiche détaillée forces/faiblesses/recommandations
6Bouton : "Réécrire cette page" → bascule vers le pipeline Éditorial avec l'audit_brief
7Export : téléchargement du rapport complet (HTML/PDF/CSV)

7. Coûts estimés

OpérationCoût
Audit d'une page (AC01-AC06 + AC08-AC10)$0.00 (100% déterministe)
Synthèse avec recommandations (AC07, LLM Haiku)~$0.005/page
Audit de 50 pages~$0.25
Rédaction optimisée (pipeline Éditorial standard)~$0.60/article
Coût quasi nul pour l'audit. Seul l'agent AC07 utilise un LLM (Haiku, le moins cher). Tout le reste est déterministe. Le vrai coût est dans la réécriture, pas dans l'audit.
