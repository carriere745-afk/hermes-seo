12. Engagement de Qualite — Les 5 Regles

1. Vrai : Les donnees affichees sont exactes ou explicitement marquees comme estimees. Pas de faux "47 positions reelles" quand GSC est absent.
2. Pertinent : Le contenu genere correspond au site analyse. Le contexte SERP est verifie avant redaction (DuckDuckGo fallback).
3. Actionnable : Chaque recommandation a un cout, un delai, une priorite, et une action concrete. Les fichiers sont generes automatiquement (sitemap, robots, llms, schemas).
4. Compréhensible : En francais, sans jargon technique excessif. Les scores sont expliques. Les limites sont annoncees.
5. Honnête : Les limites techniques sont listees (GSC non verifie, API KO). Les disclaimers juridiques sont integres. L'outil ne pretend pas remplacer un consultant.
