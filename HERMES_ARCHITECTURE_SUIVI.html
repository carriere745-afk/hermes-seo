Hermes SEO v3 — Suivi d'Architecture

Version 0.3 · 21 juin 2026 · Pipeline Rédaction ~90% · 656+ tests

1. Vue d'ensemble

Hermes SEO est un orchestrateur multi‑agents qui produit tout type de contenu (SEO / AEO / GEO) pour n'importe quelle entreprise, site web, mot‑clé et objectif.

Stack : Python 3.12 · LangGraph 1.2.5 · Streamlit · ChromaDB · SQLite · Claude Sonnet 4.6 · GPT-5.4 · Haiku 4.5 · DeepSeek V4 Flash · TalorData (SERP)

12. Méthode de travail (convenue le 21 juin 2026)

  1. L'utilisateur définit le process (quel pipeline, quels agents, quel ordre)
  2. L'utilisateur écrit le cahier des charges
  3. Claude implémente agent par agent, dans l'ordre défini
  4. Claude fait les tests (entrée valide, invalide, sortie conforme, erreur)
  5. Claude connecte au pipeline existant sans casser ce qui marche
  6. L'utilisateur valide chaque agent ou groupe d'agents
  7. L'utilisateur teste en conditions réelles
